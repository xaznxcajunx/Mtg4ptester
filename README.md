# MTG Priority Mulligan Enhanced PWA App

## Installation Instructions
1. Clone the repository using `git clone https://github.com/xaznxcajunx/Mtg4ptester.git`
2. Navigate to the project directory: `cd Mtg4ptester`
3. Install the necessary dependencies: `npm install` or `yarn install`
4. Start the application: `npm start` or `yarn start`

## Features
- **Priority Mulligan System**: Enhanced rules for selecting cards during the mulligan phase.
- **Progressive Web App**: Supports offline play and can be installed on any device.
- **User-Friendly Interface**: Simple navigation and responsive design.
- **Customizable Settings**: Configure your game according to personal preferences.

## Configuration
To configure the application:
- Open the settings menu from the main interface.
- Adjust your preferences for game rules, card visibility, and more.

## Troubleshooting
- **Issue**: Application crashes on startup.
  **Solution**: Ensure all dependencies are correctly installed. Check the console for errors.
- **Issue**: Game controls not responding.
  **Solution**: Make sure the app has permission to access your input devices.

## Game Controls
- **Start Game**: Click the 'Start' button on the main screen.
- **Mulligan Phase**: Follow on-screen instructions to select cards during the mulligan.
- **End Turn**: Click the 'End Turn' button to pass the turn to your opponent.
